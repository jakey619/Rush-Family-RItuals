# Changelog

All notable changes to this project should be recorded here.

This project uses Semantic Versioning. The current app version lives in `package.json`, release notes live in this file, and git tags map versions back to exact commits.

## [0.3.0] - Unreleased

### Added

- Added a smarter Overview dashboard with attention alerts, weekly wins, streaks, and per-person trend chips.
- Added embedded per-person action item tracking inside the app.
- Added checklist-style editing for priorities instead of fixed plain fields.

### Improved

- Extended the overview cards to show open action items, 7-day averages, and momentum.
- Kept unfinished action items carrying forward with the rest of the daily planning flow.

### Notes

- This version is not tagged yet because the changes are still in the working tree.

## [0.2.0] - Internal

- Added visible app version details in the interface.
- Added semantic version bump scripts for patch, minor, and major releases.
